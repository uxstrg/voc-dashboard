# Project Summary
The project "Simply U+" is developing a Minimum Viable Product (MVP) for LGU+ VoC (Voice of Customer) analysis using React. It facilitates the collection and analysis of customer feedback, empowering organizations to make data-driven decisions that enhance customer satisfaction and promote business growth. Recent updates include improved API integration for optimized data handling, enhancements to the user interface, and the addition of Open Graph meta tags for better social media sharing.

# Project Module Description
The project consists of several functional modules:
- **Dashboard**: Displays overall VoC metrics and trends, enhanced with new API integration for data fetching and platform-specific metrics.
- **VocFeed**: Shows detailed VoC data with filtering options, updated to utilize the new API and includes `platform` and `channel_detail` fields.
- **Guide**: Provides information on VoC attributes and definitions.
- **GNB (Global Navigation Bar)**: Navigation component for easy access to different sections of the application, recently updated to reflect the product's focus.

# Usage
To set up and run the project, follow these steps:
1. Navigate to the project directory:
   ```bash
   cd /workspace/simply-uplus
   ```
2. Remove existing dependencies:
   ```bash
   rm -rf node_modules package-lock.json
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Build the project:
   ```bash
   npm run build
   ```
